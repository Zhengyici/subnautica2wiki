# Subnautica 2 Wiki & Database

The most complete Subnautica 2 database — crafting recipes, creatures, materials, biomes, and guides.

**Live site:** https://subnautica2wiki.github.io/

## What's included

- 79 crafting recipes with full ingredient lists
- 25+ creature entries with stats and locations
- 19 material guides with where-to-find info
- 12 biome descriptions
- Beginner's guide

## Tech stack

- **Hugo** static site generator
- **GitHub Pages** free hosting
- **Fuse.js** client-side search
- **No server costs** — 100% free

## How to update content

1. Edit markdown files in `content/`
2. `git push` to main branch
3. GitHub Actions auto-deploys in ~30 seconds

## Local development

```bash
# Install Hugo: https://gohugo.io/installation/
hugo server -D
# Open http://localhost:1313
```

## Data sources

- Fandom Wiki API (subnautica2.fandom.com)
- Community research (Reddit, YouTube)
